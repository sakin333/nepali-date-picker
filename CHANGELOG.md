# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-19

### Added

- Initial release of Nepali Date Picker
- DatePicker component with BS (Bikram Sambat) and AD (Anno Domini) support
- Automatic date conversion between BS and AD calendars
- Toggle switch for switching between calendar types
- Nepali numerals support for authentic BS display
- Four color themes: blue, green, purple, red
- "Today" button for quick navigation
- Disabled state support
- TypeScript type definitions
- Utility functions: `adToBs`, `bsToAd`, `getDaysInBSMonth`, `getDaysInADMonth`, `toNepaliNum`
- Constants: `BS_MONTHS`, `AD_MONTHS`, `NEPALI_NUMERALS`
